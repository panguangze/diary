# Sample Backup Data (30 Days)

Use this sample to quickly populate the app with test content:

- **File**: `docs/sample-data/love_diary_sample_backup_30days.json`
- **Coverage**: Relationship start date `2024-12-01` with 30 consecutive mood entries (Dec 1–Dec 30) including varied mood types and example notes.

## How to Import
1. Copy the JSON file to your device/emulator (e.g., Downloads).
2. Open **设置 → 数据管理 → 导入记录** in the app.
3. Select the JSON file via the system file picker to restore it.

> 导入会覆盖当前的心情记录和配置，建议先导出现有数据作为备份。
